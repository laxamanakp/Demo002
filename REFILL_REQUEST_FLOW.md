## âœ… Approval Criteria (Case Manager Checklist)

Before approving a refill request, the Case Manager should verify:

1. **Prescription Validity**
   - [ ] Prescription is still active (not expired)
   - [ ] Prescription has remaining refills

2. **Timing Check**
   - [ ] Sufficient time has passed since last pickup
   - [ ] Patient is not requesting too early

3. **Adherence Review**
   - [ ] Patient's adherence rate is acceptable (>80%)
   - [ ] No excessive missed doses reported

4. **Inventory Check**
   - [ ] Medication is in stock at selected facility
   - [ ] Sufficient quantity available

5. **Clinical Review (if needed)**
   - [ ] No recent adverse reactions reported
   - [ ] No contraindications noted
